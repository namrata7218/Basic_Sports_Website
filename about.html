About Sports Central

Sports Central is a sports information website developed using HTML and CSS. The website is designed to provide clear and simple information about popular sports in an organized manner.

This platform includes sports such as Cricket, Hockey, and Football. Each sport has its own dedicated page containing images and basic rules, helping users understand the fundamentals of the game.

Sports Central focuses on clean design, easy navigation, and a user-friendly layout. The website serves as an educational project and helps beginners explore sports without complexity.

Website Objectives
  • To provide basic knowledge about different sports
  • To help beginners learn sports rules
  • To demonstrate HTML and CSS skills
  • To create a structured sports website
